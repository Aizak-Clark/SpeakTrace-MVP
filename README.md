# SpeakTrace — Персональный AI-тренер речи

**SpeakTrace** — это MVP (минимально жизнеспособный продукт) системы анализа качества речи, разработанный в рамках практических работ по курсу «Инструментальные средства информационных систем».

## 🚀 Функционал
Прототип позволяет пользователю анализировать текстовые расшифровки своих выступлений на наличие речевых артефактов.

- **Dashboard**: Просмотр истории сессий с ключевыми метриками (Purity Index).
- **Добавление сессий**: Возможность ручного ввода текста или загрузки файла `.txt`.
- **AI-Анализ (Имитация)**: Система автоматически распознает слова-паразиты («типа», «как бы») и паузы-хезитации («э-э»), подсвечивая их в отчете.
- **Тепловая карта**: Визуальное отображение проблемных зон в тексте отчета.
- **Хранение данных**: Локальное сохранение сессий в формате JSON.

## 🛠 Технологии
- **Backend**: Python 3.x, FastAPI.
- **Frontend**: Jinja2 Templates, Bootstrap 5 (Responsive UI).
- **Data Storage**: JSON-based file storage.
